This project uses JavaScript classes to build a simple employee management system. It includes an Employee base class, a Manager subclass that inherits from Employee using extends and super, and a Company class that stores and lists all employees. The program covers encapsulation, inheritance, and method overriding.

To run, open index.html in a browser and check the console to see the output.

Files:
index.html - Main page that links the script
cc_6.js - All class definitions and program logic
